# Java Basic

Java 기초 문법을 학습하고 실습한 내용을 기록하는 저장소입니다.

## 📚 학습 목표

- Java 기본 문법 이해
- 객체지향 프로그래밍(OOP) 기초 학습
- 자료구조 및 컬렉션 활용
- Mini MES 프로젝트(FactoryFlow) 개발을 위한 Java 기초 다지기
